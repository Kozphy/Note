# Numpy 索引，切片操作
###### tags: `python learn`

## 索引(indexing)
```python=
data = np.array([
    [1,5,2],
    [3,-1,10]
])

# 2
data[0,2]
# -1
data[1,1]
# 3
data[1,0]
```

## 資料切片
```python=
data=np.array([
    [3,1,2],[1,2,5],
    [5,2,2],[3,-5,1]
])
# [[2,2],[-5,1]]
data[2:4,1:3]

# [1,2]
data[0:2,1]


## 使用 ... 代表全都要
data=np.array([
    [
        [3,1,2],[1,0,5]
    ],
    [
        [5,4,3],[1,3,-3]
    ]
])
# [[5,4,3],[1,3,-3]]
data[1,...]
# [[[3,1],[1,0]],[[5,4],[1,3]]]
data[...,0:2]
```
