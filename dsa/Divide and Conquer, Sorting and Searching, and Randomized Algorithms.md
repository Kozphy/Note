## Karasuba Multiplication
x = 5678
y = 1234

how much is 5678 * 1234?


a = 56
b = 78
c = 12
d = 34

step 1: a * c = 672
step 2: b * d = 2652
step 3: (a+b)(c+d)= 6164
step 4: 6164 - 2652 - 672 = 2840 (ad + bc)
step 5: 672 -> 6720000, 2840-> 284000, ans: 6720000+2652+284000 = 7006652
